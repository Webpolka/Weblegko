О НАС

Переработан графический редактор блоков, упрощённая цветовая палитра административной консоли.

Улучшения в написании Gutenberg, множество вариантов стилей и расширенные возможности шаблонов для тем блоков, интегрированные шаблоны, дополнительные инструменты проектирования, возможность выбора нескольких блоков из представления списка, блокировка блоков, а также различные улучшения производительности и доступности.

Новая тема по умолчанию "Twenty Twenty-Two", новая функция редактора сайта в админке WordPress, управление темами блоков через редактор сайта, новый блок "Навигация", улучшенные элементы управления блоками, каталог шаблонов, просмотр списков, переработанный блок "Галерея", поддержка дочерних тем Theme.json, блокировка на уровне блоков, несколько таблиц стилей для каждого блока.

Новая стандартная тема Twenty Thirteen. Новый интерфейс для работы с ревизиями. Автосохранение и блокировка записей при совместном редактировании. Встроенная поддержка аудио и видео.

Переработанный интерфейс редактора сайта, улучшенный блок "Навигация", блок "Вставка блока", упорядоченная боковая панель настроек блока с вкладками "Настройки" и "Стили". Также доступна коллекция шаблонов заголовков и колонтитулов для тем блоков, интеграция с медиа и режим "Не отвлекаться" для написания текстов. Новая книга стилей дает полное представление о том, как выглядит каждый блок в библиотеке сайта, а пользователи теперь могут копировать и вставлять стили и добавлять пользовательские CSS для большего контроля над внешним видом своего сайта. Среди других возможностей - липкое позиционирование для групповых блоков верхнего уровня, возможность импорта избранных виджетов из тем Classic, а также локальные шрифты в стандартных темах WordPress для большей конфиденциальности с включением Google Fonts.
